Query string & Simple Query String

Query string

query string 类似于 URI Query

default_field： 查询字段名
query： 查询条件
and: 与操作

Simple Query String

  • 类似于Query String 只支持部分查询语法
  • 不支持 AND OR NOT 会当做字符串处理
  • Term 之间默认的关系是 or 可以指定 default_operator
  • 支持部分逻辑
      • 代替 and
      • 代替 not
    • | 代替 or

Dynamic Mapping 和常见的字段类型

什么是 Mapping

Mapping 类似于数据库中的 Schema 的定义，作用如下

  • 定义索引中的字段的名称
  • 定义字段的数据类型，eg： string int bool
  • 字段，倒排索引及分词的相关配置，

Mapping 会把 JSON 文档映射成 Lucene 所需要的扁平格式。

一个索引的Maping 属于一个索引的 Type

  • 每个文档都属于一个Type
  • 一个 Type 有一个 Mapping 定义
  • 7.0 开始，不需要在 Mapping 定义 type 的信息

字段的数据类型

简单类型

  • Text / Keyword

  • Date

  • Integer / Floating

  • Boolean

  • IPv4 / IPv6

    复杂类型

  • 对象类型/嵌套类型

    特殊类型(针对于地理信息)

  • geo_point & geo_shape / percolator
